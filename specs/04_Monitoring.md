# Choco360 活動監控規格 (Monitoring)

## 1. 監控儀表板 (Global Dashboard)
- **追蹤內容**：受評人姓名、問卷標題、目前狀態 (Pending/Ongoing/Completed)。
- **進度計算**：`已填寫人數 / (受邀人數 + 自評1人)`。

## 2. 行政干預權限 (Intervention)
- **名單管理**：管理員可針對特定邀請「新增受邀人」或「移除冗餘受邀人」。
- **移轉審核主管 (New)**：
    - 當直屬主管變動或原定主管無法審核時，管理員可在「管理名單」抽屜中修改該筆邀請的「審核主管」。
    - 變更後，系統將自動將該筆申請移至新主管的「審核名單」分頁。
- **個人進度追蹤**：在名單管理介面中，系統必須清晰標示每一位受邀人的填寫狀態（已完成/待填寫），方便管理員精準催票。
- **資料完整性限制 (Data Integrity)**：
    - **禁止移除規則**：若受邀人已完成填寫並提交反饋（顯示為「已繳卷」），行政人員**不可**將其從名單中移除。
    - **視覺表現**：對於已繳卷的人員，其移除按鈕應變更為「鎖定狀態」並顯示提示文字，以確保歷史數據不被意外刪除。
- **強制終止**：管理員有權刪除錯誤發起的評鑑週期。
- **數據重新整理**：提供一鍵同步按鈕，確保統計數據與數據庫一致。
