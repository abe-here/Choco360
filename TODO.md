# 待辦事項與路線圖 (TODO & Roadmap)

## 🛡️ 安全性與隱私提升
- [ ] **解決洩漏的 API Key 與資料庫連線資訊**：目前部分敏感資訊仍暴露在前端網頁上。這需要移至後端/伺服器端邏輯。
- [ ] **後端化安全架構 (Supabase Edge Functions)**：將 AI 分析與敏感邏輯移至 Edge Functions，隱藏 API Key 並防止 Prompt Injection。
- [ ] **隱私防護與匿名門檻**：設定最低他評門檻（如 3 份），達標後才開放查看報告，以保護回饋者的匿名性。

## 🤖 AI 與自動化整合
- [ ] **定期通知 (Slack 上的 Abe)**：實作 Slack 機器人自動提醒受邀者填寫問卷。
- [ ] **填寫時的 AI 導引**：在填寫回饋時，由 AI 即時提示使用者提供更具體、具建設性的行為證據。
- [ ] **組織架構自動化同步**：整合 Google Workspace SDK，自動同步公司組織圖與主管關係。

## 📊 報告與績效落地
- [ ] **報告下載**：新增將報告下載為 PDF 或 Excel 的功能。
- [ ] **主管審閱**：實作主管查看權限與關聯姓名功能。
- [ ] **從「診斷」轉向「行動」 (Actionable Roadmap)**：將 AI 的改善建議轉化為可追蹤的任務清單，讓員工追蹤進度。
- [ ] **基準檢測 (Benchmarking)**：在報告中增加「全公司/部門平均」參考線，提供客觀基準。
